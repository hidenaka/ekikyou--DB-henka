# ディベートコンテキスト: サービス実現の具体設計

## 合意済みの方針

### サービスの本質
- 占い風の入口 + ケースベースの意思決定支援
- 「変化の法則」ではなく「変化の型」として提示
- 予測しない。参照する
- 人が買うのは「曖昧さに耐えながら行動を決めるための納得」

### 出力の5点固定
1. あなたの現在地（局面の圧縮）
2. 今やるな（行動の禁止）
3. 今やれ（行動の推奨）
4. 反対視点（錯卦・綜卦による視点反転）
5. 類似事例（11,336件からの参照）

### ターゲット
- 漠然とした未来への不安を持つ人
- 「このままでいいのか」を整理したい人
- 順調に見える人生の中で違和感を抱える人

### 制約
- kappa=0.561のアノテーション精度
- Q6予測モデルは棄却済み
- 象徴解釈と事例参照は分離して提示
- 「成功率X%」は不可

## 既存資産
- 11,336件の実例データ（企業・個人・家族・国家）
- 八卦アノテーション付き
- 古典テキスト（卦辞・爻辞）の日本語訳
- 診断エンジン（diagnostic_engine.py）
- 易経専門家エージェント（ekikyo-expert）
- 確率テーブルによる状態→卦変換

## 既存サービス方針（life_direction_service_strategy）
- サービス名: 人生の現在地診断
- 4つの役割:
  1. 今の違和感を現在地として言語化
  2. このまま進んだ場合に起こりやすい未来の流れを見せる
  3. 別ルートの選択肢を2〜3本提示
  4. 次の30日・90日で試すべき小さな行動
- 商品形態: 現在地診断レポート
- 販路候補: ココナラ、LINE
