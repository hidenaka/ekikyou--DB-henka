# Phase 3 v2.0 同型性検証結果 - Codex批評用コンテキスト

## 前回批評からの変更点
Codex（GPT-5.2）の批判を全面的に反映してPhase 3を再設計・再実装した。

### 対応した批判
1. 「同型性」の定義不在 → 「観測遷移グラフG_obs vs Q6の構造的類似性」に明確化
2. MCA変数の情報リーク → MCA変数を一切使用しない設計に変更
3. 八卦(3bit)での検証 → 64卦(6bit)レベルでの直接検証に変更
4. 異種スペクトル比較 → G_obsラプラシアン vs Q6ラプラシアン（同種64固有値同士）
5. 棄却数ベース判定 → Fisher結合p値 + 効果量プロファイルに変更
6. 検証1のSD=0問題 → テスト自体を廃止し、Edge Overlapテストに置換

## 新設計の概要

### データソース
- `classical_before_hexagram`, `classical_after_hexagram`（64卦名、82%/81%充填）
- King Wen番号 → 6bitベクトル変換
- 有効事例数: 9,228件（2,108件は欠損で除外）

### G_obs（観測遷移グラフ）
- 64ノード（King Wen番号1-64）
- エッジ: 実データの遷移ペア（9,228件）
- 無向化して194本のユニークエッジ
- 重み付きラプラシアン行列（64x64）を構築

### 5テストの結果

| テスト | H0 | 統計量 | p値 | z/効果量 | 方向 |
|--------|-----|--------|-----|---------|------|
| A. エッジ重複率 | G_obsのQ6エッジ重複率はランダムと同等 | 0.0515 (10/194) | 1.000 | z=-3.99 large | **anti** |
| B. ハミング距離(6bit) | 遷移ペアの平均ハミング距離はランダムと同等 | 2.941 | 0.906 | z=1.36 large | **anti** |
| C. ラプラシアンスペクトル | G_obsのラプラシアン固有値とQ6のWasserstein距離はランダムグラフと同等 | W=0.431 | 0.000 | z=-4.03 large | **pro** |
| D. 錯卦対称性(64卦) | 32錯卦ペアの遷移プロファイル類似度はランダムペアと同等 | cos_sim=0.366 | 0.001 | z=3.88 large | **pro** |
| E. コミュニティNMI | G_obsとQ6のコミュニティNMIはランダム遷移と同等 | NMI=0.276 | 0.012 | z=2.69 large | **pro** |

### 帰無分布の構成（各テスト）
- Test A: after列シャッフル（周辺分布保存）→ 無向エッジ集合構築 → Q6エッジとの重複率
- Test B: after列シャッフル → 平均ハミング距離
- Test C: after列シャッフル → ランダムG_obsのラプラシアン固有値 → Q6とのWasserstein距離
- Test D: 64ノードからランダム完全マッチング（32ペア）→ コサイン類似度平均
- Test E: after列シャッフル → ランダムG_obsにLouvain → Q6コミュニティとのNMI

### 総合判定
- Fisher結合統計量: 1404.4 (df=10)
- Fisher p値: < 1e-300
- 判定レベル: **Moderate**（Fisher p<0.05 かつ pro方向small+効果3/5）

### 注目すべき点
1. **Test A（エッジ重複率）**: 実遷移のわずか5.2%（10/194）がQ6の隣接エッジ。ランダムでは9.5%。つまり実データの遷移はQ6の局所1ビット変化モデルに反する。
2. **Test B（ハミング距離）**: 平均2.94で、ランダム(2.92)とほぼ同等。遷移距離はQ6の予測（距離1偏重）と一致しない。ハミング距離の分布: {0:1113, 1:12, 2:3616, 3:78, 4:3376, 5:38, 6:995}。距離0（同一卦への遷移）が12%、距離2と4が最多。
3. **Test C（スペクトル）**: W=0.431 < ランダム0.456。効果は有意だが、差は小さい（0.025）。
4. **Test D（錯卦対称性）**: 32ペア中17ペアのみ有効（15ペアは出次数0で除外）。有効ペアの平均コサイン類似度0.366 vs ランダム0.174。
5. **Test E（コミュニティ）**: G_obsは16コミュニティ、Q6は5コミュニティ。NMI=0.276で有意だが、コミュニティ数の大きな乖離が気になる。
