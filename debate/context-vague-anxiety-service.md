# ディベートコンテキスト: 漠然とした未来への不安を整理するサービス

## 前提条件

### Q6超立方体の結論
- Q6予測モデルは完全棄却済み（0/5テスト棄却、4/5がanti方向）
- 距離1遷移の実測: 4.75%（ランダム期待値9.52%の半分以下）
- 効果量上限: -3.06%pt（ゼロ以下確定）
- 「卦Xから卦Yへの遷移確率」という予測的主張は不可能

### データベース
- 11,336件の実例データ（企業・個人・家族・国家の変化事例）
- 六十四卦の八卦構造でアノテーション済み（kappa=0.561）
- 状態→行動→結果の遷移パターンを保持

### 既存サービス方針（life_direction_service_strategy）
- サービス名: 人生の現在地診断
- ターゲット: 「順調に見える人生の中で違和感を抱え、このままでいいのかを整理したい人」
- 4つの役割:
  1. 今の違和感を現在地として言語化する
  2. このまま進んだ場合に起こりやすい未来の流れを見せる
  3. 別ルートに進む場合の選択肢を2〜3本提示する
  4. 次の30日・90日で試すべき小さな行動を示す

## ユーザーの重要な修正

### 棄却されたフレーミング
1. ❌ 「問いを生む装置」→ 人は問いを求めていない。知恵・レンズ・指針を求めている
2. ❌ 「間違った一手を減らすサービス」→ 経験者なら分かるが、若年層や未経験者は「気がついたら間違った一手を打ってしまった」ことの方が多い。事前に防ぎたいという需要が成り立たない

### ユーザーが求めるフレーミング
「もっと前段の漠然とした未来への不安感、このままいっていいのかというところを整理できるサービス。このままいったら、昔の変化の法則によったらこういうことになる可能性がある、みたいな感じ。」

### 核心的な問い
- 「漠然とした未来への不安」を持つ人に対して
- 易経の変化の法則（11,336件の実例パターン）を使って
- 「このままいったらどうなるか」の可能性を見せる
- これをどう具体的なサービスとして設計するか

## 制約
- 「成功率X%」のような予測は不可能（Q6棄却済み、kappa=0.561）
- 占い的断定はしない
- 表現体系だけでは価値が薄い
- パターン分析からの「傾向」や「典型的な展開」は提示可能
