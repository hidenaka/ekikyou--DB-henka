# Claude (Anthropic) の見解

## 改善計画v13（構造変換ノイズ+項目難易度モデル版）

### v12からの変更点
- 独立ビット反転 → **構造変換混合ノイズ（爻反転/上下取違/反転/一様）**
- 識別性アンカー → **確定ゴールド24件（専門家合議で真値固定）**
- 6爻自己回帰 → **独立6ビット（シンプル優先）**
- full overlap → **部分重複（60全+108双+108単=504注釈）**
- 注釈者能力のみ → **GLAD/IRT系（能力+項目難易度）**
- 280+20件矛盾 → **300件厳密分割（166+55+55+24）**

### 核心設計

**1. 確定ゴールド**
- 八卦×3件=24件を専門家合議で真値確定
- 学習・校正・テスト全てで使用可能

**2. 構造変換混合ノイズ**
- (1)正解 (2)単爻反転 (3)上卦取違 (4)下卦取違 (5)全体反転 (6)一様誤り
- 12×n_annotatorsパラメータで相関誤りを表現

**3. GLAD/IRT系**
- P(correct) = σ(ability × difficulty)
- 困難な項目は誰でも誤りやすい

**4. 独立6ビット予測**
- P(bits|x) = Π P(bj|x)
- 自己回帰は過剰、まずはシンプルを基準に

**5. 部分重複注釈**
- full 60件 + double 108件 + single 108件
- 総504注釈（full 828の61%で効率化）

**6. 厳密分割**
- Train 166 + Cal 55 + Test 55 + Gold 24 = 300

### 評価設計
- ベースライン: prior, majority, simple_noise, transform_no_difficulty
- 指標: NLL, Hamming, Top1/5 accuracy, ECE
- 停止条件: prior/simple_noise/transform_no_difficultyに負けたら再検討

### 工数: 200h
- Phase 1 設計: 48h
- Phase 2 ゴールド: 32h
- Phase 3 注釈: 72h
- Phase 4 モデル: 48h

### フォールバック
- 構造変換が単純に負ける → 独立ビット反転に戻す
- GLADが収束しない → 難易度無視
- α<0.3 → 上卦/下卦に簡素化
