# Claude (Anthropic) の見解

APIクレジット不足のため、Claude Codeサブエージェントが直接800件のアノテーションを実行する代替案を検討中。エージェントはcases.jsonlからテキストを読み、プロンプトテンプレートに従って4つの八卦フィールドを独立に判定する。二重注釈は2つの独立したエージェントで実行する。この方式はスクリプト方式と比べて温度制御やJSON出力の強制ができないデメリットがあるが、(1)同一モデルファミリーによるアノテーション品質は同等、(2)エージェント間のコンテキスト分離により独立性は確保される、(3)コスト面ではClaude Codeサブスクリプション内で追加課金なし、という利点がある。バッチサイズは50件/エージェントで16バッチ×2パス=32エージェント起動が現実的。
