# Claude (Anthropic) の見解

実装完了した2つの機能をレビューする。(1) /api/feedback に format=5point パラメータを追加。5layer（既定）と5point（新形式）を切り替え可能にした。5pointの場合、feedback_5pointキーに5点ビューモデルを、feedbackキーに既存5層データを両方返す（後方互換性維持）。(2) 危機ケース拒否を /api/extract と /api/diary/extract に実装。正規表現ベースで「死にたい」「自殺」「消えたい」「暴力を受け」等15パターンを検出し、専門機関の連絡先（よりそいホットライン、いのちの電話、こころの健康相談統一ダイヤル）を案内して処理を中断する。phase=crisis_rejectedでセッション状態を更新。実装はシンプルで、LLM呼び出し前の入口で検査するため、LLMコスト節約にもなる。テスト7件全パス。残タスク: 決定的事例ランキング、フロントエンド更新、品質ゲート、クローズドベータ。
