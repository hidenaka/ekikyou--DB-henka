# Phase 3 同型性検証結果 - Codex批評用コンテキスト

## 背景
易経変化ロジックDB（11,336件の実例事例）において、Q6超立方体（64卦の6次元バイナリ状態空間）と実データの統計的構造の間に同型性（isomorphism）が存在するかを検証した。

## Phase 2の結果（入力データ）
- MCA（多重対応分析）: 10次元推奨、累積寄与率55.7%（10次元時点）
- K-Means: k=15最適（silhouette=0.2205）
- 変数: before_state, after_state, trigger_type, action_type, pattern_type, scale（6変数）
- 6次元仮説: 「部分的にしか支持されない」（6次元では累積38.8%）

## Phase 3の方法
6つの統計的検証テストを実施。各テスト1,000回置換、seed=42、Bonferroni補正α'=0.00833。

## Phase 3の結果

### 総合判定: PARTIAL（部分同型性）
- H0棄却: 3/6
- 前回（旧データ5次元・k=2・13,060件）: ANALOGOUS（2/6棄却）

### 各テスト結果

| 検証 | H0棄却 | 効果量 | 方向 | 詳細 |
|------|--------|--------|------|------|
| 1. 次元数の対応 | No | negligible (d=0.000) | 中立 | MCA推奨10次元 vs Q6の6次元。置換分布の平均=27.00(SD=0.00)で検出力なし |
| 2. ハミング距離 | No | large_anti (z=2.776) | 反同型性 | 観測平均1.4913 > ランダム1.4691。実遷移はランダムより遠い八卦間 |
| 3. 八卦×クラスタ | **Yes** | medium_pro (V=0.375) | 同型性 | before_V=0.334, after_V=0.416。八卦タグがクラスタを有意に区別 |
| 4. スペクトル構造 | **Yes** | large_pro (z=-32.4) | 同型性 | Wasserstein=0.239 < ランダム0.365。MCA固有値分布がQ6に有意に近い |
| 5. 錯卦対称性 | No | medium_anti (z=-0.664) | 反同型性 | 錯卦ペア類似度0.304 < ランダム0.360。4ペアのみで検出力不足 |
| 6. 部分同型性 | **Yes** | small_pro (V=0.288) | 同型性 | 八卦×パターンタイプのCramer's V。DBSCAN k=2 vs Q6コミュニティ5は不一致 |

### レポートで挙げた限界
1. 八卦（3ビット）レベルの分析であり、64卦（6ビット）レベルの完全な遷移情報ではない
2. 検証1の置換テストでMCA完全再計算の代わりにSVD近似を使用
3. 検証5は4ペアのみで統計的検出力が不足
4. Phase 2のMCA Dim1-2の不安定性（r≈0.25）が影響する可能性
5. DBSCAN k=2のロバスト性が低い

## 質問
1. PARTIAL判定は妥当か？3/6棄却だが、pro方向のlarge効果量は検証4のみ。
2. 検証4のz=-32.4は異常に大きいが、surrogate_mcaベースラインの妥当性は？
3. 検証2のlarge_anti（遷移が遠い八卦間で起きる）は、易経の理論（変化は対極へ向かう）と整合する可能性があるが、同型性テストでは反同型性と評価される。この解釈は妥当か？
4. 全体として、Q6モデルの実用性についてどう評価するか？
