# Claude (Anthropic) の見解

知恵エンジンの価値を3層で整理する。第1層「名前がつく」価値：人が最も苦しいのは「何が起きているか分からない」時。モヤモヤに名前がつくだけで楽になる。易経は3000年かけて変化の全パターンに名前をつけた体系。「あなたの状況は屯（産みの苦しみ）です」と言われた瞬間、混乱が整理される。これはカウンセリングの概念化と同じ機能。第2層「やるな」の価値：人は何をすべきかより何をすべきでないかの方が実は重要。間違った行動を止められるだけで被害は半減する。易経の384爻は各段階での禁忌を具体的に示す。これは予測ではなく、3000年の失敗事例から蒸留された知恵。第3層「見えていない角度」の価値：確証バイアスにより人は自分の見たい情報しか見ない。錯卦は正反対の視点、綜卦は相手の視点、互卦は隠れた内部構造を強制的に提示する。これは体系的にできる点でChatGPTの汎用アドバイスより優れている。ジョブ理論で言えば：ユーザーのジョブは人生の岐路で判断の質を上げたい。代替手段との比較：占いは断定するが根拠がない。コーチングは高い。自己啓発本は汎用的すぎる。ChatGPTは体系がない。友人は専門性がない。知恵エンジンは体系的で根拠があり安いというポジション。金を払う瞬間：転職、起業、離婚、大きな投資、キャリアの岐路。人生に数回しかない大きな判断の前。継続価値：月1回の現在地チェック。ゆでガエルにならないための定点観測。不可欠性テスト：正直、無くなっても困らない人が多い。だからこそ一度使って驚く体験の設計が重要。初回体験で自分の状況がこんなに正確に言語化されるのかと感じさせることが勝負。
