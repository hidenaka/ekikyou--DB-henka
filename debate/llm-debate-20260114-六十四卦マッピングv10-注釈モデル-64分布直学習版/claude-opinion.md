# Claude (Anthropic) の見解

## 改善計画v10（注釈モデル+64分布直学習版）

### v9からの変更点
- 6爻独立分解を廃止 → 64卦分布を直接学習
- w(r)集約を廃止 → 注釈モデル（EM推定）でsoft label生成
- 目標値なしを廃止 → ベースライン比較+停止条件を明示
- c_uモデル都合調整を廃止 → coverage-risk曲線を出し意思決定者が選択

### 核心設計

**1. 注釈モデル（ラベル生成）**
- 潜在変数zを置き、注釈者ごとの混同行列（温度・バイアス）を推定
- EMアルゴリズムで事後分布 q(z|annotations) を計算
- 「特定2〜3卦に集中する」相関パターンを保持

**2. 予測モデル（64分布直学習）**
- 注釈モデルの事後分布を教師として64クラス交差エントロピー
- 相関した不確実性を直接学習
- スケールによる上限なし

**3. 構造化オプション（フォールバック）**
- 上卦/下卦の階層（8×8）: P(上卦|x) × P(下卦|x, 上卦)
- 独立積ではなく条件付きで相関を表現

**4. 意思決定**
- c_uは運用要件から固定
- coverage-risk曲線を意思決定者に提示し選択させる

**5. 評価設計**
- ベースライン必須: Prior only、単純64クラス、上卦/下卦独立
- 停止条件: 全ベースラインに勝てなければ設計再検討
- 不確実性: Bootstrap 95%CIで報告
