# 抽象テーマ動画テンプレート（3ブロック構成）

## 改善ポイント反映版
- ✅ 1秒辞典を冒頭に配置
- ✅ 3ブロック構成でテンポアップ（Hook→反復→分岐）
- ✅ 統計表現を「行動示唆」に調整（断定を避ける）
- ✅ URL誘導は1回のみ（Outro）
- ✅ サウンド設計を統一化

---

## 構成概要（60秒）

| ブロック | 秒数 | 内容 |
|---------|------|------|
| **1秒辞典** | 1-2秒 | 今回使う八卦の読みと意味を瞬時に提示 |
| **Hook** | 3-5秒 | 統計＋問いかけ（断定しない表現） |
| **反復** | 25-30秒 | 昭和/平成/令和を圧縮して「同じパターンの反復」を見せる |
| **分岐** | 15-20秒 | 未来A（失敗）vs 未来B（成功）を対比 |
| **Outro** | 3-5秒 | 問いかけ＋診断URL（1回のみ） |

---

## テンプレート詳細

### ブロック0: 1秒辞典（1-2秒）

**目的**: 視聴者が八卦の読みと意味を瞬時に把握

```
視覚:
- 画面中央に大きく表示
- 「☵ 坎（かん）＝ 試練」
- 「☰ 乾（けん）＝ 強さ」
- フェードイン/アウト 各0.5秒

音声: なし（または短い効果音のみ）

テロップのみで伝える
```

**バリエーション**:
- パターンA: 2つの八卦を同時表示（変化を示す場合）
- パターンB: 1つの八卦のみ（感情・状況を示す場合）

---

### ブロック1: Hook（3-5秒）

**目的**: 統計で興味を引く＋問いかけで自分ごと化

**統計表現の改善（断定を避ける）**:

❌ NG表現（断定的すぎる）:
- 「82.7%が成功する」
- 「これが法則だ」
- 「必ずこうなる」

✅ OK表現（行動示唆・傾向提示）:
- 「678件で最も多かった変化はこれ」
- 「82.7%がこのパターンを辿った」
- 「データが示す、ひとつの傾向」

**台本例**:
```
視覚:
- 八卦シンボルの変化アニメーション（坎☵ → 乾☰）
- 背景色が青から金へグラデーション

ナレーション:
「678件の変化の中で、
 このパターンは81%が、
 強さに辿り着いた」

テロップ: 坎 → 乾｜49件中81%

音声設計:
- 坎の瞬間: 水滴音、低音
- 乾への変化: 上昇音、金属音
```

**統計表現の整合性ルール**:
- 「678件」= 全データベース件数（権威性を出す）
- 「このパターンは XX%」= 個別パターンの統計
- 必ず「678件の中で、このパターンは」と関係を明示する
- テロップには「49件中81%」のように母数を表示

---

### ブロック2: 反復（25-30秒）

**目的**: 「歴史は繰り返す」を圧縮して見せる

**改善**: 昭和/平成/令和を「同じパターン」として重ねる

**構成**:
- 各時代5秒以内
- 同じ八卦シンボルと色で統一
- 「また同じことが起きた」感を出す

**台本例（坎→乾パターン）**:
```
--- 昭和（5秒）---
視覚:
- モノクロの焼け野原（抽象的）
- 背景: 坎の青（#4169E1）
- シンボル: ☵（右上に固定表示）

ナレーション:
「昭和、戦後。
 誰もが『坎』にいた」

音声設計: 水滴音、重い低音

--- 平成（5秒）---
視覚:
- バブル崩壊の抽象イメージ
- 同じ坎の青
- 同じ☵シンボル

ナレーション:
「平成、バブル崩壊。
 また『坎』が来た」

音声設計: 同じ水滴音（反復を強調）

--- 令和（5秒）---
視覚:
- パンデミック、空っぽの街
- 同じ坎の青
- 同じ☵シンボル

ナレーション:
「令和、パンデミック。
 三度目の『坎』」

音声設計: 同じ水滴音（反復を強調）

--- 反復の結論（5秒）---
視覚:
- 3つの時代が重なるイメージ
- 坎☵ → 乾☰ への変化

ナレーション:
「そして毎回、
 同じ変化が起きた」

音声設計: 水滴音 → 上昇音へ変化
```

---

### ブロック3: 分岐（15-20秒）

**目的**: 未来A/Bの対比で「選択」を意識させる

**改善**: 統計は「傾向」として提示

**台本例**:
```
--- 未来A（7-10秒）---
視覚:
- 暗闘に沈む人影（抽象的）
- 坎☵ → さらに暗い色
- または 坎☵ → 震☳（混乱へ）

ナレーション:
「逃げた場合、
 データでは
 6.7%しか抜け出せなかった」

テロップ: 逃避 → 6.7%

音声設計: 低音がさらに重く

--- 未来B（7-10秒）---
視覚:
- 光が射す、立ち上がる人影
- 坎☵ → 乾☰
- 背景が青から金へ

ナレーション:
「だが向き合った場合、
 81%がここに辿り着いた。
 『乾』。強さ」

テロップ: 向き合う → 81%

音声設計: 上昇音、金属音、解放感
```

---

### ブロック4: Outro（3-5秒）

**目的**: 診断への誘導（1回のみ、シンプルに）

**改善**: URLは1回だけ、押し付けない

**台本例**:
```
視覚:
- 8つの八卦シンボルが円形配置
- 中央に「あなたは今、どこ？」
- 下部に診断URL（小さめ、目立ちすぎない）

ナレーション:
「あなたは今、どこにいる？」

テロップ: 診断 → [URL]

音声設計: 余韻を残す音（問いかけの間）
```

**NG**: 「今すぐ診断！」「絶対見て！」などの押し付け

## サウンド設計統一ルール

| 八卦 | 基本音 | 変化時の音 |
|------|--------|------------|
| 乾☰ | 金属音、高音 | 上昇音、輝く音 |
| 坤☷ | 大地の低音 | 安定音、着地音 |
| 震☳ | 雷鳴、衝撃音 | 破裂音、警告音 |
| 巽☴ | 風の音、流れ | 柔らかい上昇音 |
| 坎☵ | 水滴、深い低音 | 沈む音、重い音 |
| 離☲ | 火の音、パチパチ | 輝く音、発火音 |
| 艮☶ | 静寂、無音 | 止まる音、息を止める |
| 兌☱ | 明るい音、水の跳ね | 笑い声、解放音 |

**変化の音**:
- ネガティブ変化: 下降音、重くなる
- ポジティブ変化: 上昇音、軽くなる
- 中立: フェードイン/アウト

## 使用方法

1. このテンプレートをコピー
2. テーマに合わせて八卦を選択
3. `hexagram_1sec_dictionary.json` から読み・意味を取得
4. 統計データを `pattern_guide.json` から取得
5. 台本を具体化
6. ローカル動画環境で生成
