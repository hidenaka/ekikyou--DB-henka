# 易経のビジネス応用フレームワーク

易経は単なる占いではなく、変化の法則を体系化した「意思決定の科学」としてビジネスに応用可能です。以下に主要な応用フレームワークを定義します。

## 1. 6段階の成長とタイミング（爻の位とキャリア/プロジェクト）

| 爻位 | ステージ | ビジネスにおける意味 | 推奨アクション |
|------|----------|----------------------|----------------|
| **初爻** | **潜龍 (Hidden Dragon)** | **準備・潜伏期**<br>市場調査、スキル習得、チームビルディング、水面下の活動。まだ表に出る時期ではない。 | `Invest` `Learn` `Plan`<br>拙速なリリースを避け、基盤を固める。 |
| **二爻** | **見龍 (Seeing Dragon)** | **デビュー・参入期**<br>MVPリリース、テストマーケティング、現場での実践。有力なメンターやパートナー（大人）に出会うべき時。 | `Launch` `Connect`<br>現場での成果を出し、信頼を勝ち取る。 |
| **三爻** | **君子終日乾乾** | **試練・激動期**<br>急成長に伴うリスク、競合の出現、内部摩擦。昼夜を問わず努力し、警戒を怠らなければ咎なし。 | `Iterate` `Persist`<br>批判やトラブルに誠実に対応し、改善を繰り返す。 |
| **四爻** | **或躍 (Leaping)** | **飛躍のチャンス・岐路**<br>市場でのポジショニング確立、ピボットの決断。深淵に留まるか、天に躍るかの選択。 | `Decision` `Pivot`<br>タイミングを見計らい、大胆に勝負に出る（または退く）。 |
| **五爻** | **飛龍 (Flying Dragon)** | **全盛期・リーダーシップ**<br>シェアNo.1、業界のリーダー、CEOの視点。利をもって大人を見る（優れた人材・パートナーを活用する）。 | `Lead` `Delegate`<br>ビジョンを掲げ、組織を統率し、成果を最大化する。 |
| **上爻** | **亢龍 (Arrogant Dragon)** | **衰退・引退期**<br>市場飽和、組織の硬直化、老害化。進むだけで退くことを知らないと悔いあり。 | `Exit` `Succession`<br>後継者に道を譲る、新規事業へリソースを移す、撤退戦。 |

## 2. 意思決定マトリクス（Go / No-Go）

相談内容に対して、得られた卦・爻からアクションを推奨する基準。

- **Go (攻める・拡大)**
  - 乾、震、離、大有、晋、升 などの「上昇・明るい」卦
  - 二爻、五爻が得位（中正）である場合
  - **戦略**: 積極投資、新規参入、シェア拡大

- **Wait / Prepare (待機・守備)**
  - 坤、艮、需、屯 などの「停止・準備・難産」卦
  - 初爻、三爻の場合
  - **戦略**: 足元を固める、人材育成、内部留保の蓄積

- **Stop / Retreat (撤退・損切り)**
  - 遯、剥、蹇、困 などの「衰退・困難」卦
  - 上爻、または凶の変爻
  - **戦略**: 撤退、事業売却、コスト削減、リスク回避

- **Pivot / Change (転換・改革)**
  - 革、鼎、復、過 などの「変革・刷新」卦
  - **戦略**: 組織再編、ビジネスモデルの転換、リブランディング

## 3. 組織論とリーダーシップ

- **三才（天・地・人）の適用**
  - **天 (Top Management)**: ビジョン、戦略、タイミングの把握（五爻・上爻）
  - **地 (Resources/Field)**: 現場、リソース、顧客対応（初爻・二爻）
  - **人 (Middle Management)**: 調整、オペレーション、コミュニケーション（三爻・四爻）

- **陰陽のバランス**
  - **陽（Active）**: イノベーション、営業、拡大志向。多すぎると暴走・過労。
  - **陰（Passive）**: リスク管理、総務・経理、安定志向。多すぎると停滞・保守。
  - 組織編成において、陽のリーダー（乾）には陰の補佐役（坤）を配置するなど、補完関係を意識する。

## 4. リスク管理（4大難卦の対応）

ビジネスにおけるクリティカルな状況とその対応。

1. **水雷屯 (Difficulty at the Beginning)**
   - **状況**: 創業期の混乱、産みの苦しみ。
   - **対策**: 焦って動かず、協力者を求め（利建侯）、基盤を作る。

2. **坎為水 (The Abysmal)**
   - **状況**: 資金繰りの悪化、裏切り、連続するトラブル。
   - **対策**: 誠実さ（孚）を持ち、小事を行う。無理に脱出を焦らず、耐え抜く。

3. **水山蹇 (Obstruction)**
   - **状況**: 行き詰まり、プロジェクトの座礁。
   - **対策**: 西南（平坦な道）に行けば吉、東北（険しい道）は凶。偉大な人物に会う（利見大人）。専門家に頼る。

4. **沢水困 (Oppression)**
   - **状況**: 資源枯渇、精神的疲弊、八方塞がり。
   - **対策**: 言葉を信じてもらえない時期。寡黙に、致命傷を避ける（命を致し志を遂ぐ）。
