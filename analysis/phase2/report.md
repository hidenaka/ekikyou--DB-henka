### 15. k=2 の解釈と限界

k=2の分離は **カテゴリの頻度構造** を反映しており、意味論的に異なる遷移パターンを発見したものではない。
MCAは低頻度カテゴリを原点から遠い位置に配置する性質があるため、これらのカテゴリを含む事例は
5次元座標空間で自然にクラスタ化される。

**結果として**: k=2は「スキーマ準拠 vs 非準拠」の二分割であり、
遷移のメカニズムや質的な差異に基づく分類ではない。

### 17. 遷移パターンの体系

データが示す変化の法則性:

1. **成功の安定化**: 安定成長・成功 → 安定・平和 (46.4%, z=+27.5) — 成功は安定に収束する
2. **V字回復**: どん底・危機 → 持続成長・大成功 (37.3%, z=+10.4) — 底打ちからの反転は頻出
3. **慢心の代償**: 絶頂・慢心 → 崩壊・消滅 (28.8%, z=+13.7) — 慢心は崩壊を招く
4. **安定の脆弱性**: 安定・平和 → 崩壊・消滅 (4.9%, z=-18.0) — 安定から直接崩壊はまれ
5. **混乱の収束困難**: 混乱 → 持続成長 (8.4%, z=-17.0) — 混乱から直接の大成功は困難
