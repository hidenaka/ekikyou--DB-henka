## 致命的問題（CRITICAL）

### C1: マルコフ連鎖分析の状態空間不整合

**深刻度**: CRITICAL — 定常分布・混合率・吸収状態の全解釈が無効

**問題の詳細**:

遷移行列は before_state（7状態）× after_state（8状態）の非正方行列。before_stateとafter_stateは**異なる状態空間**であり、重複は3状態のみ:

| before_stateのみ (4) | 共通 (3) | after_stateのみ (5) |
|--------------------|---------|-------------------|
| どん底・危機 | 停滞・閉塞 | 変質・新生 |
| 成長痛 | 安定・平和 | 崩壊・消滅 |
| 混乱・カオス | 安定成長・成功 | 持続成長・大成功 |
| 絶頂・慢心 | | 混乱・衰退 |
| | | 縮小安定・生存 |

`transition_matrix.py` の line 244:
```python
P = prob_matrix.reindex(index=states, columns=states, fill_value=0).values
```

この行はbefore_states (7) を行・列の両方に使って7×7正方行列を構築する。しかし:
- after_stateにしか存在しない5状態（変質・新生, 崩壊・消滅, 持続成長・大成功, 混乱・衰退, 縮小安定・生存）の列が欠落
- before_stateにしか存在しない4状態（どん底・危機, 成長痛, 混乱・カオス, 絶頂・慢心）の列はfill_value=0で埋められる

**結果として生じた行列P**:
- 行和 ≈ 0.06〜0.59（1.0ではない）→ **確率的行列（stochastic matrix）ではない**
- 「安定成長・成功」行: 停滞0.003 + 安定・平和0.464 + 安定成長0.125 = 0.592 (残り0.408が消失)
- 「どん底・危機」行: 停滞0.063 + 安定・平和0.068 + 安定成長0.128 = 0.259 (残り0.741が消失)

**固有値分解への影響**:
- 最大固有値 ≈ 0.357（本来の確率的行列なら1.0であるべき）
- `np.argmin(np.abs(eigenvalues - 1.0))` は1.0に最も近い固有値を探すが、実際には0.357を選択
- 正規化後の「定常分布」は数学的に固有ベクトルの正規化であり、確率的解釈を持たない
- 報告された second_eigenvalue=0.082, mixing_rate=0.918 は sub-stochastic 行列の第2固有値であり、「高速混合」の解釈は**完全に無効**

**定常分布の集中（安定・平和52.7%, 安定成長・成功31.1%, 停滞・閉塞16.1%）はアーティファクト**:
- この3状態はbefore/afterの両方に存在する唯一の状態
- 行和が0でない列を持つ唯一の状態
- 他の4状態（どん底・危機, 成長痛, 混乱・カオス, 絶頂・慢心）の列は全て0 → 定常分布が0になるのは必然

**修正方針**:
1. before_state と after_state が異なる状態空間であることを明示的に認め、マルコフ連鎖分析を「不適用」と記録する
2. あるいは、before_state と after_state を統一した状態空間で遷移データを再構築した上で、改めてマルコフ分析を実施する（ただし、データの性質上これが適切かの判断が必要）
3. レポート (`report.md`) のマルコフ連鎖セクション（定常分布、混合率、吸収状態）を修正または撤回する

## 軽微な問題（MINOR）

### m1: 列座標不安定性のPhase 3への伝播リスク

Dim1 r=-0.225, Dim2 r=0.266 という列座標の不安定性は、MCA空間上のクラスタリング結果全体の信頼性に疑問を投じる。raw dataの5次元を使用する判断は合理的だが、不安定な列座標に基づくMCA空間でのクラスタリングがPhase 3の同型性検証に耐えうるかは慎重な評価が必要。

Gap統計量が k=15 を示す一方、シルエットは k=2 を示す。この大きな乖離（15 vs 2）は、データの構造がシンプルな距離ベース指標と理論的な参照分布ベース指標で全く異なる解を示すことを意味する。レポートでの議論が不十分。
