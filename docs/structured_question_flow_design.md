# 構造化質問フロー設計書 v1.0

## 概要

六十四卦診断サービスの核心となる「構造化質問方式」の詳細設計。
乱数を使わず、ユーザーの回答から決定論的に卦を算出する。

## 設計原則

1. **決定論**: 同じ回答 → 同じ卦（100%再現可能）
2. **易経準拠**: 八卦の本質的意味に基づく質問設計
3. **reading_id固定**: 診断結果はIDに紐付けて永続保存
4. **無料/有料一貫性**: 卦の算出結果は同一、付加価値で差別化

## フェーズ1: 視点とテーマの固定

### Q1: 視点の固定
```
誰の立場で診断しますか？

A) 自分自身
B) 組織・会社
C) 家族・チーム
D) 第三者（観察対象）
```
→ 後続の質問文脈を調整（「あなた」「御社」「そのチーム」等）

### Q2: 主テーマの固定
```
何について診断しますか？

A) キャリア・仕事
B) 人間関係
C) 事業・プロジェクト
D) 健康・生活
E) 学習・成長
F) 財務・資産
G) その他（自由記述）
```
→ 類似事例検索のフィルタに使用

## フェーズ2: 上卦の決定（外的状況）

上卦 = 外部環境・相手・客観的状況を表す

### Q3: エネルギーの方向性
```
現在の外部環境は、どちらに近いですか？

A) 拡大・発展・上昇の傾向
B) 収縮・停滞・下降の傾向
```
→ A: 陽系（乾/離/震/兌）、B: 陰系（坤/坎/艮/巽）

### Q4: 動静の状態
```
外部環境の動きは？

A) 活発に動いている
B) 静かに留まっている
```
→ A: 動卦（震/巽/坎/離）、B: 静卦（乾/坤/艮/兌）

### Q5: 象徴的イメージ
```
外部環境を象徴するなら？

【拡大×動】
A) 雷のような突発的変化・衝撃
B) 炎のような情熱・注目・明晰さ

【拡大×静】
C) 天のような強大な力・権威
D) 沢のような喜び・交流・開放

【収縮×動】
E) 風のような浸透・じわじわ広がる影響
F) 水のような困難・試練・深み

【収縮×静】
G) 地のような受容・安定・従順
H) 山のような停止・境界・瞑想
```

| 拡大 | 静 | C | 乾（天） |
| 拡大 | 静 | D | 兌（沢） |

## フェーズ3: 下卦の決定（内的状況）

下卦 = 自分自身・主体・主観的状態を表す

### Q6: 内的エネルギーの方向性
```
あなた自身の現在の状態は？

A) 積極的・能動的・前向き
B) 消極的・受動的・慎重
```
→ A: 陽系、B: 陰系

### Q7: 内的な動静
```
あなたの行動傾向は？

A) 動きたい・変化を起こしたい
B) 待ちたい・様子を見たい
```
→ A: 動卦、B: 静卦

### Q8: 内的象徴イメージ
```
自分自身を象徴するなら？

【積極×動】
A) 雷：決断力・行動力・突破
B) 火：明晰さ・情熱・表現

【積極×静】
C) 天：強い意志・リーダーシップ
D) 沢：喜び・柔軟性・コミュニケーション

【慎重×動】
E) 風：柔軟な対応・浸透・適応
F) 水：忍耐・深い思考・困難への対処

【慎重×静】
G) 地：受容・サポート・安定志向
H) 山：熟考・境界設定・内省
```

## フェーズ4: 爻位の決定（進展段階）

爻 = 状況の進展段階（1〜6）

### Q9: 時間軸での位置
```
この状況の進展段階は？

A) 始まったばかり・萌芽期
B) 初期段階・まだ形になっていない
C) 中間・転換点に差し掛かっている
D) 後半・形が見えてきた
E) 終盤・まとめの段階
F) 完了間近・次への移行期
```

### 爻位決定マッピング
| 選択 | 爻位 | 意味 |
|------|------|------|
| A | 初爻 | 潜龍。まだ動くべきでない |
| B | 二爻 | 田に現る。徐々に姿を現す |
| C | 三爻 | 危険な転換点。慎重に |
| D | 四爻 | 跳躍の準備。機を待つ |
| E | 五爻 | 中正の位。最も良い位置 |
| F | 上爻 | 極点。亢龍に注意 |

### 類似事例検索ロジック
```
1. 同卦（上卦+下卦が一致）を抽出
2. 同爻（active_lineが一致）でフィルタ
3. theme（main_domain）でさらに絞り込み
4. 最大5件を類似度順で返却
```

## 無料版 vs 有料版の出力差

### 無料版出力
- 卦名 + 象徴（1行）
- 爻位の基本解説（2-3行）
- 類似事例1件の概要

### 有料版出力
- 卦の詳細解説（卦辞・象伝）
- 爻辞の詳細解説
- 状況分析（LLMによる当てはめ）
- 類似事例3-5件の詳細
- 分岐シナリオ（吉/凶の条件）
- 具体的行動計画
