# 人生の現在地 — 意思決定支援サービス 商品定義書 v3

**作成日**: 2026-03-09
**ステータス**: 確定（Phase 0）

## 1. サービス定義

### 1.1 何であるか
- 「順調に見える人生の中にある違和感」を構造化し、選択肢の比較と次の一歩を整理する**意思決定支援サービス**
- 実例データベース（11,336件）からの類似事例パターンに基づく**シナリオ提示**
- 人間レビューを経た個別レポートの納品

### 1.2 何でないか
- 未来予測サービスではない
- 占い・スピリチュアルサービスではない
- 心理カウンセリング・医療行為ではない
- 転職斡旋・求人紹介ではない
- 「正解」を教えるサービスではない

### 1.3 シナリオ提示の定義
- 「予測」ではなく「類似事例からのパターン提示」
- 各シナリオには以下のラベルを必ず付与:
  - **観測済み**: アウトカムが確認された事例に基づく
  - **仮説**: 類似パターンからの推論であり、観測済みアウトカムではない
- 確率・的中率等の数値は一切使用しない

## 2. 想定顧客

### 2.1 ターゲット
- 外から見ると順調だが、本人の内側に「このままでいいのか」がある
- 転職・独立・移住など、結論がまだ決まっていない
- 何かを変えたい気はあるが、何から動けばいいか分からない
- 視点と順番が欲しい

### 2.2 非ターゲット
- 仕事紹介・求人提案を期待する人
- 心身の不調が強く、医療・心理支援が先に必要な人
- 娯楽として占いを楽しみたい人
- 自分で動く意思がなく、正解だけを受け取りたい人

## 3. 提供物（6セクションレポート）

| セクション | 内容 | 根拠ラベル |
|-----------|------|-----------|
| 1 現在地の構造化 | 違和感の言語化、状況の整理 | — |
| 2 このまま進んだ場合のシナリオ | 類似事例からの延長パターン | 観測済み/仮説 |
| 3 別ルート 2-3本 | 異なる選択をした類似事例からのパターン | 観測済み/仮説 |
| 4 各ルートの比較表 | 向き・リスク・必要行動・前提条件の比較 | — |
| 5 今日の1アクション | 最小の具体的行動 | — |
| 6 30日/90日実験プラン | 小さな実験の設計 | — |

## 4. 差別化の主語（外向き）
- 「実例ベースのケース比較 + 比較表 + 実験プラン + 人間レビュー」
- 易経は**外向きには出さない**。内部の分類語彙としてのみ使用
- 確率・的中率・予測は一切使わない

## 5. 禁止事項
- 「あなたの未来は〇〇です」等の断定的予測
- 確率・パーセンテージの提示
- 易経・八卦・卦名のユーザー向け表示
- 占い・スピリチュアル用語の使用
- 医療・法律・金融の専門的助言
- 「正解」「最適解」「ベストな選択」等の表現
