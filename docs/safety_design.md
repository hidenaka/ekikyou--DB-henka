# 安全設計仕様書（Safety Design Specification）

**作成日**: 2026-03-09
**ステータス**: 確定（Phase 0）
**目的**: 意思決定支援サービスにおける危機ケースの検知・対応・改善の運用設計

## 1. 3層検知アーキテクチャ

### Layer 1: キーワード検知（即時・自動）
入力テキストに対してキーワードマッチングを実行。

#### 検知カテゴリとキーワード
| カテゴリ | キーワード例 | 重要度 |
|---------|------------|--------|
| 自傷・自殺 | 死にたい, 消えたい, 自殺, 首を吊る, 飛び降り, リストカット, OD, 楽になりたい | CRITICAL |
| 他害 | 殺したい, 殴りたい, 復讐, 報復 | CRITICAL |
| DV・虐待 | 殴られる, 暴力, 虐待, 監禁, 怖い人 | HIGH |
| 依存 | アルコール依存, 薬物, ギャンブル依存, やめられない | HIGH |
| 重度抑うつ | 何もできない, 布団から出られない, 食べられない, 眠れない(2週間以上) | MEDIUM |
| 経済的危機 | 借金, 破産, 闇金, 取り立て | MEDIUM |

#### 注意事項
- キーワードは文脈無視で発火するため、偽陽性が発生する
- Layer 1はフラグ付与のみ。遮断判断はLayer 2以降

### Layer 2: LLM分類（文脈理解・自動）
Layer 1でフラグが付いた入力、または全入力に対してLLMで文脈分類を実行。

#### 分類プロンプト（概要）
```
以下のテキストを読み、危機レベルを分類してください。
- CRITICAL: 即時の生命危険がある
- HIGH: 専門家の介入が必要
- MEDIUM: 注意が必要だが、サービス提供は可能（制限付き）
- LOW: 危機兆候なし
- FALSE_POSITIVE: キーワードは含むが、危機的文脈ではない

分類根拠を1文で説明してください。
```

#### 判定ルール
| LLM判定 | アクション |
|---------|----------|
| CRITICAL | 即時遮断 → 緊急テンプレート送信 → 人間通知 |
| HIGH | サービス一時停止 → 専門窓口案内テンプレート → 人間確認待ち |
| MEDIUM | サービス継続（制限付き）→ 注意フラグをレビューに付与 |
| LOW | 通常処理 |
| FALSE_POSITIVE | 通常処理（ログに記録） |

### Layer 3: 人間最終判断
- CRITICAL/HIGH判定時は、必ず人間（サービス提供者）が最終判断
- 判断結果をログに記録
- 判断に迷う場合は安全側に倒す（サービス停止 + 窓口案内）

## 2. 対応テンプレート

### 2.1 CRITICAL（即時生命危険）
```
【重要なお知らせ】
お伝えいただいた内容について、私たちのサービスでは適切な対応ができません。
今すぐ、以下の窓口にご連絡ください。

■ いのちの電話: 0120-783-556（毎日16時〜21時/毎月10日は8時〜翌8時）
■ よりそいホットライン: 0120-279-338（24時間対応）
■ こころの健康相談統一ダイヤル: 0570-064-556

緊急の場合は、110番（警察）または119番（救急）にお電話ください。

お気持ちを打ち明けてくださったことに感謝します。
専門のスタッフがあなたのお話を聞いてくれます。
```

### 2.2 HIGH（専門家介入推奨）
```
お伝えいただいた内容について、私たちのサービスよりも
適切な専門家のサポートをお勧めしたいと思います。

以下の窓口で、無料で相談できます:
■ よりそいホットライン: 0120-279-338（24時間対応）
■ [該当カテゴリに応じた専門窓口]

当サービスは意思決定の整理をお手伝いするものですが、
今のご状況では、まず専門家にお話しされることをお勧めします。
準備が整いましたら、いつでもご利用いただけます。
```

### 2.3 MEDIUM（注意付きサービス継続）
```
ご状況を伺いました。
当サービスは意思決定の整理をお手伝いするもので、
医療・心理・法律の専門的な助言は行っておりません。

必要に応じて、以下の窓口もご活用ください:
■ [該当する相談窓口]

その上で、意思決定の整理としてお手伝いできる範囲で
レポートを作成いたします。
```

## 3. ログ・記録・改善ループ

### 3.1 記録項目
全ケース（危機フラグの有無に関わらず）:
- 入力テキスト（匿名化済み）
- Layer 1フラグ結果
- Layer 2 LLM分類結果
- Layer 3 人間判断結果（該当時）
- 最終アクション
- タイムスタンプ

### 3.2 事後レビュー
- 週次: 全フラグケースのレビュー（誤検知/見逃しの確認）
- 月次: キーワードリスト・LLMプロンプトの更新
- 四半期: 安全設計全体の見直し

### 3.3 改善ループ
1. 誤検知（FALSE_POSITIVE）が多いキーワード → 文脈条件を追加
2. 見逃し（事後に判明）→ キーワード追加 + LLMプロンプト更新
3. 対応テンプレートへの不満 → テンプレート改訂

## 4. 免責・ディスクレーマー

### サービス利用規約に含める文言
- 当サービスは医療・心理・法律・金融の専門的助言を提供するものではありません
- 提示するシナリオは過去の類似事例に基づく参考情報であり、将来の結果を保証するものではありません
- 深刻な心身の不調がある場合は、専門家にご相談ください
- 当サービスの利用により生じた損害について、責任を負いかねます

## 5. スコープ制限

### サービスが扱うテーマ
- キャリア・働き方の方向性
- 生活環境の変化（移住、住み替え等）
- 人間関係の整理（家族、パートナー等の大きな変化）
- 学び・スキルの方向性
- ライフステージの変化への対応

### サービスが扱わないテーマ
- 医療的判断（治療方針、服薬等）
- 法律的判断（離婚手続き、相続、訴訟等）
- 金融的判断（投資、資産運用等）
- 特定の企業・人物への評価
