# 八卦アイコン画像生成プロンプト

## 共通指示（全アイコン共通）

```
あなたは易経（I Ching）の八卦をモダンなアイコンとしてデザインするデザイナーです。

【デザインスタイル】
- モダンでミニマルなフラットデザイン
- 円形のアイコンベース（app icon style）
- 微細なグラデーションで立体感を演出
- 背景は透過または単色
- 512x512px

【必須要素】
1. その卦を象徴する自然物・シンボル（メイン）
2. 伝統的な三本の爻線（━━━ or ━ ━）をアイコン下部に小さく配置
3. 卦に対応するカラーパレット

【デザイン原則】
- 易経に馴染みのない人でも直感的に意味が伝わる
- 日本・中国の伝統要素と現代的デザインの融合
- シンプルだが印象的
- 動画素材としても映える
```

## 1. 乾（けん）- 天 / Heaven

```
【生成プロンプト】

Create a modern minimalist icon for "Qian" (乾), the Heaven trigram from I Ching.

Visual elements:
- A radiant golden sun or celestial sphere at the center
- Subtle rays of light emanating outward
- Alternatively: a stylized dragon head or royal crown silhouette
- Three solid horizontal lines (━━━) at the bottom representing yang-yang-yang

Color palette:
- Primary: Royal gold (#D4AF37) to deep purple (#663399) gradient
- Accent: White light effects
- Background: Deep navy or transparent

Mood: Majestic, creative, powerful, celestial
Style: Flat design with subtle gradients, circular icon format, 512x512px

The icon should convey: Leadership, creativity, strength, the ultimate yang energy
```

## 一括生成用プロンプト（8つまとめて）

```
【まとめて生成する場合】

易経（I Ching）の八卦（8つの基本シンボル）をモダンなアプリアイコンスタイルでデザインしてください。

8つのアイコンセット：
1. 乾（天）- 金色、太陽/龍、全て陽線
2. 兌（沢）- 水色、湖/笑顔、上が陰
3. 離（火）- 赤橙、炎/目、中が陰
4. 震（雷）- 黄色、稲妻/足、下だけ陽
5. 巽（風）- 緑、風/木、下だけ陰
6. 坎（水）- 紺、深水/月、中だけ陽
7. 艮（山）- 茶、山/手、上だけ陽
8. 坤（地）- 黄土、大地/母、全て陰

【スタイル統一要素】
- 円形アイコン（512x512px）
- フラットデザイン＋微細グラデーション
- 各アイコンの下部に伝統的な三本の爻線を小さく配置
- 背景透過
- モダンだが東洋的な雰囲気
- 一目で意味が伝わるシンボル

【用途】
- 易経診断アプリの結果表示
- YouTube動画の説明素材
- SNS共有用画像
```
