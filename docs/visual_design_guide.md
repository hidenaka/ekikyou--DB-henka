# 易経ビジュアルデザインガイド

## 概要

易経変化ロジックDBの診断結果を視覚的に分かりやすく表現するためのデザインシステム。
易経に馴染みのない人でも直感的に理解できることを目指す。

## デザイン原則

1. **伝統と現代の融合**: 易経の本質を尊重しつつ、現代的で洗練されたデザイン
2. **直感的理解**: シンボルを見ただけで意味が伝わる
3. **一貫性**: 全アイコンで統一されたスタイル
4. **汎用性**: 白背景・黒背景どちらでも映える、動画素材としても使用可能

## 八卦アイコンデザイン仕様

### 共通仕様

- **サイズ**: 512x512px（マスター）、256x256px、128x128px、64x64px
- **形状**: 円形ベース（八卦の循環性を表現）
- **スタイル**: フラットデザイン + 微細なグラデーション
- **線の太さ**: 統一感のある中太線
- **背景**: 透過PNG

## アイコン構成要素

各アイコンは以下の要素で構成：

```
┌─────────────────────┐
│                     │
│    [シンボル]       │  ← 自然物や抽象形状
│                     │
│    ━━━━━━━         │
│    ━━━━━━━         │  ← 伝統的な爻線（小さめ）
│    ━━ ━━           │
│                     │
└─────────────────────┘
```

## 六爻（段階）の視覚表現

爻位を示すための補助ビジュアル：

| 爻位 | 段階 | 視覚メタファー | 色温度 |
|-----|------|---------------|-------|
| 初爻 | 始まり | 種、芽 | 淡い緑 |
| 二爻 | 発展 | 成長する苗 | 緑 |
| 三爻 | 転換点 | 分岐する道 | 黄色（警告） |
| 四爻 | 新段階 | 扉、階段 | 青 |
| 五爻 | 中心 | 王冠、星 | 金 |
| 上爻 | 極み | 山頂、満月 | 紫/銀 |

## 使用場面

1. **診断結果カード**: 本卦・之卦などの表示
2. **変化の流れ図**: 矢印と組み合わせて変化を表現
3. **YouTube動画素材**: 説明動画の視覚補助
4. **SNS共有用画像**: 診断結果のシェア

例: hexagram_03_chun_512.png

line_position_{number}_{size}.png
