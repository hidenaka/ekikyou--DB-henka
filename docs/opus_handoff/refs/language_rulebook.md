# HaQei 言葉遣いルールブック

## 基本トーン

**寄り添い × 温かい言い切り**

| 要素 | 説明 |
|------|------|
| **寄り添い** | 相手の状況を否定しない、選択肢を提示する |
| **温かい言い切り** | 断定はするが、「提案」として伝える（命令ではない） |
| **弱さの開示** | 完璧な答えを持っていないことを認める |

**イメージ**: 信頼できる友人が、自分の経験と失敗談をもとに話してくれる感じ

---

## 🔥 トーン設計の3層構造（NEW）

成功事例（しいたけ占い、POSIWILL、北欧暮らしの道具店）から学んだ構造：

```
【Layer 1: 肯定・安心】
「今は休んでいい時期かもしれません」
「焦らなくて大丈夫」
　　　↓
【Layer 2: 代弁・言語化】
「『このまま続けていいのかな』という感覚、ありますよね」
「それって、こういうことかもしれません」
　　　↓
【Layer 3: 提案・選択肢】
「〜という見方もあります」
「〜してみてもいいかもしれません」
```

**ポイント**:
- Layer 1 で安心させてから、Layer 2 で言語化、Layer 3 で提案
- いきなり「〜すべき」は使わない

## 肯定と安心のフレーズ集（しいたけ占い参考）

### 基本フレーズ

| フレーズ | 使用場面 |
|---------|---------|
| 「今は待っていい時期です」 | 焦っている人へ |
| 「焦らなくて大丈夫」 | 決断を急いでいる人へ |
| 「十分頑張ってきたと思います」 | 自己否定している人へ |
| 「それは自然な感覚です」 | 不安を感じている人へ |
| 「今のあなたの判断は、間違っていないと思います」 | 確認を求めている人へ |

### 「〇〇してもいい」形式

| フレーズ | ニュアンス |
|---------|----------|
| 「休んでもいい時期です」 | 許可を与える |
| 「迷っていてもいいんです」 | 焦りを和らげる |
| 「まだ決めなくてもいいと思います」 | 判断を急がせない |
| 「立ち止まってもいい季節です」 | 季節のメタファー |

### 避けるべき：フワッとした慰め（NG）

| NG | 理由 |
|----|------|
| 「大丈夫、きっとうまくいく」 | 根拠がない |
| 「なんとかなりますよ」 | 無責任に聞こえる |
| 「あなたなら大丈夫」 | 具体性がない |

**代わりに**: 根拠を添える（「9,000件のデータでは〜」「このパターンでは〜」）

## 代弁のフレーズ集（POSIWILL参考）

### 「代弁」とは

> ユーザーが言葉にできないモヤモヤを、こちらが言語化してあげること。
> 「初めて理解された」という感覚を生み、強い信頼関係を築く。

### 代弁の型

| 型 | 例文 |
|----|------|
| 「〜という感覚、ありますよね」 | 「『このまま続けていいのかな』という感覚、ありますよね」 |
| 「〜って、こういうことかもしれません」 | 「そのモヤモヤって、こういうことかもしれません」 |
| 「〜という人、多いです」 | 「『情報は揃ってるのに決められない』という人、多いです」 |
| 「言葉にすると〜」 | 「言葉にすると『変わりたいのに変われない』ですよね」 |

### ペルソナ別・代弁フレーズ

#### ユウタ（キャリア転換期）向け

| モヤモヤ | 代弁 |
|---------|------|
| 何となく違和感 | 「『自分じゃなくてもいい仕事』をしている感覚、ありますよね」 |
| 転職の迷い | 「『動いた方がいいのか、待った方がいいのか』分からない、ですよね」 |
| 焦り | 「周りが活躍しているのを見ると、焦りますよね」 |
| 失敗への恐怖 | 「『また同じ失敗をするんじゃないか』という不安、ありますよね」 |

#### ミサキ（スピリチュアル志向）向け

| モヤモヤ | 代弁 |
|---------|------|
| 占いの限界 | 「『確かにその面はある、でもそれが全てじゃない』って感じますよね」 |
| 抽象的なアドバイス | 「『で、具体的に何をすればいいの？』と思うこと、ありますよね」 |
| 状況による変化 | 「『状況によって自分は変わるのに』と感じること、ありますよね」 |

## 弱さの開示フレーズ集（北欧暮らし参考）

### なぜ「弱さの開示」が重要か

> 「完璧なカリスマではなく、自分たちの痛みを理解してくれる『隣人』からの提案を受け入れる」
> — 市場調査レポートより

### 開示の型

| 型 | 例文 |
|----|------|
| 「僕も〜した」 | 「僕も変化に失敗したことがある」 |
| 「正直〜」 | 「正直、まだ分からない部分もある」 |
| 「〜かもしれない」 | 「これが正解かは分からないかもしれない」 |
| 「〜と言われている」 | 「3000年前からこう言われている」（自分の意見ではなく） |

### 具体的な開示フレーズ

| フレーズ | 使用場面 |
|---------|---------|
| 「僕も衝動で動いて失敗したことがある」 | 転職・決断の話 |
| 「変わろうとして、逆に自分を見失ったことがある」 | 変化の話 |
| 「完璧な答えは持っていない」 | 複雑な状況の話 |
| 「これは一つの見方であって、正解かどうかは分からない」 | 解釈の話 |
| 「一緒に考えていきたい」 | 結論を急がない姿勢 |

### 避けるべき：過度な謙遜（NG）

| NG | 理由 |
|----|------|
| 「私なんかが言うのもなんですが」 | 信頼性が下がる |
| 「素人考えですが」 | 価値が下がる |
| 「間違ってるかもしれませんが」 | 毎回言うと信頼されない |

**バランス**: 弱さは見せるが、価値は下げない

## 使う表現（OKリスト）

### 主語・視点

| OK | 例 |
|----|-----|
| 「僕は」「私は」 | 「僕はこう思う」 |
| 「3000年前の人たちは」 | 「3000年前の人たちはこれを知っていた」 |
| 「変化には」 | 「変化にはパターンがある」 |
| 「データを見ると」 | 「9,000件のデータを見ると」 |

### 結論の言い方（温かい言い切り）

| OK | 例 | ニュアンス |
|----|-----|----------|
| 「〜だと思う」 | 「今は待った方がいいと思う」 | 提案 |
| 「〜らしい」 | 「待った方がいいらしい」 | 伝聞 |
| 「〜かもしれません」 | 「今は待つ時かもしれません」 | 可能性 |
| 「〜という見方もあります」 | 「こういう見方もあります」 | 選択肢 |
| 「〜という傾向があります」 | 「76%が好転するという傾向がある」 | データ |
| 「〜してみてもいいかもしれません」 | 「試してみてもいいかもしれません」 | 提案 |

### 共感フレーズ

| OK | 用途 |
|----|------|
| 「転職しようか迷った時」 | キャリア層向けフック |
| 「変わりたいのに変われない」 | 両ペルソナ向け共感 |
| 「情報は揃ってる、でも決められない」 | 意思決定の葛藤 |
| 「占いに物足りなさを感じる」 | スピリチュアル層向け |
| 「モヤモヤしている」 | 汎用 |
| 「何となく違和感がある」 | 汎用 |

### 価値提示フレーズ

| OK | 用途 |
|----|------|
| 「3000年前の知恵」 | 権威付け（易経と言わない） |
| 「変化のパターン」 | コアコンセプト |
| 「こうなりやすい」 | 占いとの違い |
| 「待つべき時、動いていい時」 | 実用的価値 |
| 「9,000件以上で検証」 | データ根拠 |

## 使わない表現（NGリスト）

### 専門用語

| NG | 理由 | 言い換え |
|----|------|----------|
| 「易経」 | ハードルが上がる | 「3000年前の知恵」「古典」 |
| 「卦（け/か）」 | 意味不明 | 「パターン」 |
| 「爻（こう）」 | 意味不明 | 「段階」「フェーズ」 |
| 「変爻」 | 意味不明 | 「変化のポイント」 |
| 「乾坤離坎」 | 意味不明 | 使わない |

### 占いっぽい表現

| NG | 理由 |
|----|------|
| 「予言」「宿命」 | スピリチュアルすぎる |
| 「当たる」「外れる」 | 占いの評価基準 |
| 「運勢」「運気」 | 神秘的すぎる |
| 「占ってみた」 | 占いとの区別がつかなくなる |

### 煽り・押し付け

| NG | 理由 |
|----|------|
| 「今すぐ」「期間限定」 | ブランドトーンに合わない |
| 「このままだと失敗する」 | 恐怖を煽る |
| 「まだ迷っているんですか？」 | 攻撃的 |
| 「〜しないとダメ」 | 押し付け |
| 「絶対に」「必ず」 | 断定しすぎ |
| 「残りあと〇名」 | 欠乏訴求 |

### 上から目線

| NG | 理由 |
|----|------|
| 「教えてあげる」 | 上から |
| 「分かっていない」 | 批判的 |
| 「間違っています」 | 否定的 |
| 「〜すべき」 | 押し付け |
| 「〜しなさい」 | 命令 |

## トーン調整の具体例（拡充版）

### Before → After

| Before（NG） | After（OK） | 変換ポイント |
|-------------|-------------|-------------|
| 「易経の渙卦では」 | 「3000年前の本には」 | 専門用語回避 |
| 「今すぐ転職すべき」 | 「動いてもいい時期らしい」 | 提案型 |
| 「このままだと失敗する」 | 「待った方がいいかもしれない」 | 恐怖→提案 |
| 「占いの結果」 | 「パターンで見ると」 | 占い回避 |
| 「私が教えます」 | 「一緒に見ていこう」 | 上から→寄り添い |
| 「変わらないとダメ」 | 「変化にはタイミングがある」 | 押し付け→事実 |
| 「あなたは今〇〇です」 | 「今は〇〇という時期かもしれません」 | 断定→可能性 |
| 「迷うな」 | 「迷っていてもいいんです」 | 命令→許可 |
| 「行動しろ」 | 「行動してみてもいいかもしれません」 | 命令→提案 |
| 「正解はこれ」 | 「こういう見方もあります」 | 正解→選択肢 |

### 冷たい → 温かい変換

| 冷たい | 温かい |
|--------|--------|
| 「待つべきです」 | 「待ってもいい時期だと思います」 |
| 「動くべきではない」 | 「今は動かなくても大丈夫かもしれません」 |
| 「失敗パターンです」 | 「気をつけた方がいいポイントがあります」 |
| 「間違った選択です」 | 「別の見方もあるかもしれません」 |

## チェックリスト（動画・投稿前）

### 基本チェック

- [ ] 「易経」「卦」等の専門用語を使っていない
- [ ] 「当たる/外れる」ではなく「こうなりやすい」
- [ ] 煽り表現（今すぐ、このままだと）を使っていない
- [ ] 上から目線になっていない

### 温かさチェック（NEW）

- [ ] **肯定・安心**の要素が入っている（「〜してもいい」「焦らなくて大丈夫」）
- [ ] **代弁**の要素が入っている（「〜という感覚、ありますよね」）
- [ ] **提案型**になっている（「〜すべき」ではなく「〜してみてもいいかも」）
- [ ] **弱さの開示**が自然に入っている（「僕も〜」「正直〜」）
