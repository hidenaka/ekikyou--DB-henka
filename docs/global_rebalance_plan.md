## 追加事例ポリシー

### 1. ソース要件

国際事例は以下のソースを必須とする:

| Tier | ソース例 | 備考 |
|------|----------|------|
| tier1 | 各国政府、SEC、EU機関 | 最優先 |
| tier2 | Reuters, Bloomberg, FT, NYT, BBC | 推奨 |
| tier4 | 企業公式IR/プレスリリース | 許可 |

### 2. ドメインホワイトリスト追加

国際事例追加に伴い、以下を追加:

**tier1_official**:
```
sec.gov, europa.eu, ecb.europa.eu
fca.org.uk, bafin.de, amf-france.org
```

**tier2_major_media**:
```
guardian.com, spiegel.de, lemonde.fr
scmp.com, straitstimes.com
```

**tier4_corporate**:
```
samsung.com, tsmc.com, alibaba.com
siemens.com, volkswagen.de, lvmh.com
```

### 3. 優先事例カテゴリ

| カテゴリ | 件数目標 | 理由 |
|----------|----------|------|
| Fortune Global 500企業 | 200件 | 高品質ソース豊富 |
| テック/スタートアップ | 150件 | 変化事例が多い |
| M&A/経営統合 | 100件 | 変化パターン明確 |
| 危機対応/再建 | 100件 | Failure事例確保 |
| 歴史的事例 | 50件 | 長期視点 |
