### サービス内容

```
【こんなお悩みありませんか？】

✓ 仕事も生活も順調なはずなのに、なぜかモヤモヤする
✓ 「このままでいいのか」が頭から離れない
✓ 転職、独立、移住...考えてはいるけど、何から動けばいいか分からない
✓ 選択肢が多すぎて、逆に決められない
✓ 周りに相談しても「贅沢な悩み」と言われてしまう

【このサービスでできること】

あなたの「違和感」を丁寧に聞き取り、状況を構造的に整理します。
その上で、11,000件以上の実例データベースから類似の状況にあった人たちのパターンを調べ、
あなたの選択肢を比較するレポートをお届けします。

■ レポートの内容（全6セクション）

1. 現在地の整理 — あなたの状況を言語化します
2. このまま進んだ場合 — 類似事例から見える傾向
3. 別のルート2-3本 — 異なる選択をした人たちのパターン
4. ルート比較表 — 各選択肢のリスク・条件・必要行動を一覧で比較
5. 今日の1アクション — 5-15分でできる最小の一歩
6. 30日/90日実験プラン — 小さく試す設計

■ 特徴

・実例ベース: 11,000件以上の実例データベースから類似事例を検索
・比較表付き: 感覚ではなく、条件を並べて比較
・実験設計: 「正解を選ぶ」のではなく「小さく試す」アプローチ
・人間レビュー: AIだけでなく、人の目で確認してお届けします

【何をしないか】

✗ 未来を予測すること
✗ 「正解」を教えること
✗ 転職先や求人を紹介すること
✗ 医療・法律・金融の専門的な助言

【ご利用の流れ】

① ご購入後、現在の状況やお気持ちを自由にお書きください（200字以上）
② 内容の確認のため、1-2回やりとりさせていただく場合があります
③ 48時間以内にレポートをお届けします
④ 7日後に簡単なフォローアップをお送りします

【お願い】

・心身の不調が強い場合は、先に医療機関や専門の相談窓口をご利用ください
・「正解」を知りたい方には向きません。ご自身で考え、動く意思がある方向けのサービスです
```
